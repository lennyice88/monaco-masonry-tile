# Monaco Masonry & Tile — Website

Marketing website for **Monaco Masonry & Tile LLC**, a family-owned masonry and tile contractor
based in Medford, MA, serving Massachusetts, New Hampshire, and Rhode Island.

## Stack

Plain **HTML / CSS / JS** — a self-contained single-page site with client-side page switching and
no build step. Uses Google Fonts (Fraunces + Inter) and Lucide icons via CDN.

## Run it

Just open `index.html` in any browser. No server or install required.

## Structure

- `index.html` — markup for all pages (Home, Services, Gallery, Book, Quote, Contact)
- `style.css` — light editorial luxury design system (black / gold / cream / white)
- `script.js` — page switching, gallery, forms, micro-interactions
- `photos/` — logo, hero, owner photo, and project gallery images
